CheckSumTool

CheckSumTool is an utility program for calculating and verifying checksums for files. The tool supports CRC32, MD5 and SHA1 -checksums in its initial version.

CheckSumTool is open source software. It is licenced under MIT Licence. The MIT Licence allows everybody to download the sources, modify sources, compile and distribute them. Hovever licence text must be always distributed with the software.

Go to project pages at SourceForge.Net.
